# SilverCare Meal Planner · 银龄膳养助手

SilverCare Meal Planner 是一个集中展示老年营养饮食方案的单页网页。逻辑、样式和数据全部打包在 `index.html` 中，打开即可体验居民端、食堂端和配送端三种视角，了解同一份菜谱如何满足不同角色的需求。

## 模块概览
- **居民端**：填写身高、体重、季节、慢病、忌口等信息，系统实时计算 BMI、提示能量/蛋白范围，并组合主菜、配菜、汤饮与茶饮/水果搭配。
- **食堂端**：强调油盐与营养统计，呈现食材换算、注意事项和批量出餐提示，便于制定配餐表。
- **配送端**：聚焦配送批次、到餐时间、保温方式等字段，可直接改写成配送清单或播报模板。

## 特色功能
1. **健康评分**：针对高血压、糖尿病、血脂异常等慢病设置盐、油、糖、嘌呤、GI 等阈值，并对菜谱打分、列出扣分理由。
2. **季节菜库**：内置 16+ 道主菜、配菜与汤饮，附“痛风慎”“糖友慎”等标签，支持按季节与病种筛选。
3. **茶饮/水果建议**：根据季节与慢病组合推荐饮品与水果盘，可快速生成“今日茶饮”“低糖组合”等提示。
4. **三餐联动**：记录早餐后，午晚餐能量与蛋白配额自动扣减；数据保存在浏览器本地存储中。
5. **界面呈现**：暖色卡片式排版搭配 SVG 插画和提示条，便于课堂、路演或社区宣讲展示。

## 数据自定义
- `DEFAULT_DISHES`、`DEFAULT_SIDE_DISHES`、`DEFAULT_SOUPS` 控制主菜、配菜、汤饮数据，按 JSON 结构增删即可。
- `DEFAULT_RULES` 管理评分权重、油盐目标与慢病阈值，可直接改写参数。
- `TEA_SUGGESTIONS`、`FRUIT_SUGGESTIONS`、`SEASONAL_TEA_RECIPES` 为茶饮/水果提示，可替换为本地化配方。
- BMI 分段、能量拆分比例、餐次文案等逻辑也集中在同一文件，统一调整即可。

## 使用建议
- 进入页面后先补充身高/体重与慢病信息，右侧 BMI 卡片会提示当日能量、蛋白范围。
- 切换食堂端或配送端，可呈现同份菜谱在不同角色下的关注点，适合讲解流程或做演示。
- 需要清空当天记录时，可在界面内使用“清空记录”按钮；否则刷新仍保留数据。

## 常见问题
- **样例菜谱从哪里来？** 目前采用整理过的社区营养建议，用于演示流程，可替换为营养师审核菜单。
- **免责声明够用吗？** 页脚已有“仅供饮食参考，不替代医疗建议”，可根据使用场景另行补充隐私或授权说明。
- **能接后台吗？** 可以。如需实时数据，可在脚本中调用 `fetch`，把静态常量替换为 API 响应。
